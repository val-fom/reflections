Q: When would you want to use a remote repository rather than keeping all your work local?

A: When I'm waking on different computers, in team and/or want to share my code with somebody

Q: Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?

A: Because while I'm working in my project I want to know what exactly is going on with it

Q: Describe the differences between forks, clones, and branches. When would you use one instead of another?

A: Forks are placed on GitHub only. Clone can be like a copy of remote repository on local machine, a both on local. Branches is a label used to mark some commits for different experimets or developing some new features.
